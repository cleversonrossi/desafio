### Tecnologias Usadas:
* JavaScript
* Typescript 2.3.4
* Bootstrap 3.3.32
* HTML

### Para executar:
* Instalar angular:
```sh
  npm install -g @angular/cli@1.4.9
```
* Instalar dependencias do projeto:
```sh
  npm install
```
* Iniciar o projeto:
```sh
  npm start
```
* Acessar projeto em:
  - http://localhost:4200/
