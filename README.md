# Tecnologias:
* JavaScript
* Typescript 2.3.4
* Bootstrap 3.3.32
* HTML

# Para executar:
* Instalar angular: npm install -g @angular/cli@1.4.9:
* Instalar ferramenta de deploy: npm install -g surge
* Instalar dependencias do projeto: npm install
* Iniciar o projeto: npm start
* Acessar projeto em:
* * http://localhost:4200/
